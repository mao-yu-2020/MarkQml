# 侧颜与花

> 在光与影交织的瞬间，侧脸是最美的轮廓。

---

## 关于这幅画

这是一幅充满**梦幻感**的侧脸肖像。画面中的少女低垂眼眸，发丝间点缀着*粉色的花朵*，仿佛时间都在此刻静止。

## 技术细节

渲染这幅画时，需要关注以下 `shader` 参数：

```cpp
// 光斑效果配置
float bokehIntensity = 0.8;
vec3  bloomColor    = vec3(0.9, 0.85, 0.95);
```

## 视觉元素

画面包含丰富的层次：

- **发丝**：细腻的线条与渐变
- **花朵**：点缀在耳侧的粉色装饰
- **光斑**：背景中散景效果的圆形光点
- **色调**：柔和的蓝绿与暖黄交织

## 绘制流程

1. 勾勒侧脸的基本轮廓
2. 铺设整体的冷暖色调
3. 细化发丝与花朵的细节
4. 添加背景的光斑与氛围
